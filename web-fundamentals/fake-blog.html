Basic Keyboard Sizes:

Size Description
Fullsize/100% Fullsize keyboards come with the traditional alpha keys, numbers, modifiers, arrow keys, and numpad.
Tenkeyless/80% Tenkeyless keyboards remove the numpad from traditional fullsize keyboards.
Compact/60% Compact keyboards, additionally omit the arrowkeys and function row present in tenkeyless keyboards.
65% 65% keyboards are 60% keyboards with an additional column to allow for arrow keys.
75% 75% keyboards are 65% keyboards with the function row.
40% 40% keyboards are 60% keyboards with no number row.
